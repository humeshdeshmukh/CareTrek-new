## Typography Hierarchy

```
Header Title
Font: 20px, Bold
Color: Primary text

Metric Value
Font: 28px, Bold
Color: Metric color (red, green, etc.)

Metric Label
Font: 13px, Medium
Color: Secondary text

Stats
Font: 16px, Semi-bold
Color: Primary text

Tab Text
Font: 12px, Medium
Color: Accent (active) / Secondary (inactive)
```

## Spacing System

```
Padding:
- Card: 16px
- Tab: 8px vertical, 12px horizontal
- Header: 8px top, 16px horizontal

Margins:
- Between cards: 12px
- Between tabs: 0px (space-around)
- Bottom padding: 24px

Border Radius:
- Cards: 16px
- Icons: 16px
- Tabs: 0px (bottom border only)
```

## Shadow & Elevation

```
Card Shadow:
- Offset: 0px, 2px
- Opacity: 0.1
- Radius: 4px
- Elevation: 3 (Android)

Tab Shadow:
- Offset: 0px, 1px
- Opacity: 0.05
- Radius: 2px
- Elevation: 1 (Android)
```

## Animation Timings

```
Tab Switch: 300ms (smooth)
Card Press: 100ms (quick feedback)
Opacity Change: 200ms (smooth)
```

## Accessibility Features

```
Color Contrast:
- Text on background: 4.5:1 (WCAG AA)
- Text on cards: 4.5:1 (WCAG AA)
- Icons: 3:1 minimum

Touch Targets:
- Minimum size: 48x48px
- Cards: 60px+ height
- Tabs: 44px+ height

Text:
- Minimum size: 12px
- Line height: 1.5x
- Letter spacing: 0.5px (labels)
```
